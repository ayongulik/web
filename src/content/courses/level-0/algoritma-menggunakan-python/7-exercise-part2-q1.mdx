# Exercise Part 2 - Q1

Buatlah kode python yang menerima input berupa bilangan bulat positif dan mencetak pola segitiga siku-siku menggunakan simbol `*`. Sebagai gambaran, di bawah ini adalah output yang diinginkan apabila program menerima input `6`:

```
*
**
***
****
*****
******
```

Gunakan template ini untuk mengerjakan soal.

```python
def main(lines):
    # tulis kode kamu disini

if __name__ == "__main__":
    # tes input yang berbeda disini
    main(6)
```

<Alert variant="tip">

Secara default, fungsi `print` akan menambahkan newline atau `\n` pada setiap akhir string yang dicetak. Kita dapat mengubah sifat tersebut dengan mengeset opsi `end` saat melakukan pemanggilan fungsi `print` seperti berikut:

```python
print('Hello world!', end='') # Hasil cetak akan diakhiri dengan string kosong
```

</Alert>
